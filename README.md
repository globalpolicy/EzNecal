# EzNecal
Easy Nepali calendar for windows, made with Electron.
<p>
Has the following features:
<ul>
<li>Run on startup</li>
<li>Add/remove border</li>
<li>Bold toggle</li>
<li>Highlight on hover</li>
<li>Topmost toggle</li>
</ul>
</p>
